# Taper Pulley Calculator

[![Open Tool](https://img.shields.io/badge/🌐_Open_Tool-Online_Calculator-blue?style=for-the-badge)](https://volkovhl.github.io/TaperPulley-Calc/)

**Taper Pulley Calc** — веб‑калькулятор для инженеров, механиков и DIY‑энтузиастов. Он помогает рассчитать ключевые параметры посадочного конуса для шкива за секунды.

## 🧮 Что умеет калькулятор

- **Расчет геометрии конуса**: вычисление углов, глубин и диаметров посадочного места.
- **Мгновенные результаты**: экономит время при подборе или проектировании шкивов.
- **Визуализация параметров**: удобное отображение входных данных и итоговых значений.
